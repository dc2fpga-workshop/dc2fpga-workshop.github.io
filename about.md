## About the Workshop

Welcome to the **DC<sup>2</sup>-FPGA: 1st Workshop on FPGA Systems for Data Centers and Cloud**. 

Today, FPGAs are widely adopted across various computing systems, especially in cloud data centers. Thanks to their performance advantages and reconfigurable nature, FPGAs have been exploited for diverse computing and application domains, where they serve as not only coprocessor-like accelerators similar to GPUs/NPUs, but also as SmartNICs for high-speed networking, quantum controllers for real-time error correction, and as CXL memory/storage controllers for building cache-coherent distributed systems.

This workshop aims to discuss the state-of-the-art and future of FPGA computing stacks at cloud scale, from hardware to software and applications, across multiple computing domains—including computer architecture, high-performance computing (HPC), operating systems, programming language, and quantum computing. The workshop consists of three parts: (1) A hands-on introduction and tutorial to Coyote, an open-source HW/SW system stack for data center FPGAs, (2) invited talks about multi-domain cloud FPGA systems, and (3) a technical session for talks from researchers, students, and engineers working on relevant research domains. 

## Call for Talks
We invite submissions for talks on a broad range of FPGA-related research, from application-specific hardware acceleration to software/runtime support for FPGAs.
The topics include, but are not limited to: 
- FPGA adoption and orchestration in cloud data centers
- FPGA-accelerated cloud applications (Machine Learning/LLM, graph processing, data analytics, etc.)
- Middleware, OS, and hypervisor support for FPGAs
- FPGA shells and hardware support
- Network systems using FPGAs
- Memory and storage systems using FPGAs 
- FPGA integration into quantum computing
- Cloud FPGA security

We invite submissions and participation from the entire community.
